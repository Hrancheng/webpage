My Projects

Android app with RNN for voice identification

This is an Android app we developed with RNN model, able to identify Quran verses with the users' recitation. We use Python package to transform voice information to Numpy array. And trained the model with peak loss smaller than 0.1. We collected ground-truth data from Youtube recitation videos. We are still working to improve the accuracy of the model and publish the app to Google Play Store.
